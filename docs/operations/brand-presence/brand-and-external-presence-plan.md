## Purpose

This document defines the early SearchClarity brand asset and external presence plan.

It exists so branding, social profiles, video channels, and third-party entity signals are not forgotten while the first offer, website, Fiverr launch assets, and sample report are being built.

This is SearchClarity-owned.

It is not Neon Ronin doctrine.

---

## Core Principle

```text
SearchClarity.co is the canonical home.
External channels reinforce the brand entity and route trust back to the website.
```

SearchClarity should not try to post everywhere.

The goal is a small, serious, consistent external footprint.

## Short-Form Video Distribution Model

YouTube should be the primary video/proof/methodology hub.

Instagram Reels and TikTok should be short-form distribution channels, not separate content machines.

Recommended pipeline:

```text
SearchClarity.co topic or service question
-> short educational script
-> record one vertical video
-> publish as YouTube Short
-> republish/adapt to Instagram Reels
-> republish/adapt to TikTok
-> summarize on LinkedIn when useful
-> link back to SearchClarity.co where appropriate
```

Acceptable video types:

- 30-60 second educational shorts
- sample report walkthrough clips
- common Etsy seller mistake breakdowns
- no-guarantee SEO expectation clips
- human-reviewed audit methodology clips
- keyword-dump vs real audit comparisons

Avoid:

- daily posting obligations before the business is ready
- trend-chasing that does not support SearchClarity
- generic AI content sludge
- unsupported ranking/sales claims
- using customer examples without consent

## Entity Footprint Rules

External profiles should use consistent:

- brand name
- logo/avatar
- short bio
- website link
- service language
- no-guarantee positioning
- human-reviewed/evidence-backed framing
- same canonical domain
- same founder/company relationship once decided

Do not let external profiles describe SearchClarity differently enough that the brand entity becomes fuzzy.

## Launch Order

Recommended order:

1. Lock basic brand language.
2. Create logo/avatar direction.
3. Reserve priority handles.
4. Create LinkedIn Company Page.
5. Align founder LinkedIn profile.
6. Reserve YouTube channel.
7. Reserve X/Twitter, Instagram, TikTok handles.
8. Create banners/profile images.
9. Build SearchClarity.co MVP pages.
10. Add YouTube/video only when there is proof/methodology content worth publishing.
11. Republish shorts to Instagram and TikTok once a reusable video template exists.

---

## Not Current Priorities

Do not focus on:

- daily posting schedules
- managing every channel independently
- turning SearchClarity into a media company before the offer is proven
- making Substack the main content home
- creating platform-specific content calendars before the website/service proof exists
- public customer examples without consent

## Exit Criteria

This workstream is ready enough when:

- basic brand asset requirements are defined
- logo/avatar/banner needs are known
- priority external channels are selected
- handles are reserved or intentionally parked
- profile bio language is consistent
- external profiles link back to SearchClarity.co
- short-form video republishing model is understood
- no external channel is treated as canonical over SearchClarity.co
