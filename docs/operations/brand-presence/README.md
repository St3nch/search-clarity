# Brand Presence Operations

## Purpose

This folder contains SearchClarity branding, visual identity, social profile, and external presence planning.

Use this folder for brand assets, profile setup, banners, bios, external channel strategy, short-form video distribution, and entity-footprint guidance.

This is SearchClarity-owned material.

It is not Neon Ronin doctrine.

## Operating Rule

SearchClarity.co is the canonical home.

External channels should reinforce the SearchClarity entity and send trust/signals back to the owned website.

Do not create social channels that SearchClarity cannot maintain or intentionally park.
