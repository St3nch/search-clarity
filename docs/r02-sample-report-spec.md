## 1. Executive Summary

SearchClarity launches with no reviews, no client results, and no track record. The sample report is the only proof asset available at launch. It needs to do the job that testimonials normally do — immediately signal to a buyer that this is a serious, professional service worth paying for.

**The buyer fear to overcome:** "This is just another $15 Fiverr gig that spits out a list of tags I could have found myself." The sample report must prove that is not what they are getting.

### What the sample report must prove

| Proof point | How the report shows it |
|---|---|
| Human-reviewed, not auto-generated | Reasoning is shown, not just conclusions. The "how we reached this" section is present. Writing sounds like a person, not a prompt output. |
| Specific to the shop, not generic | Every recommendation references the actual (fictional) listing, title, or tag being reviewed. No generic advice that could apply to anyone. |
| Evidence-based | Sources and observations are named. Tool data is labeled as estimated. Manual observations are described. |
| Honest about limits | Caveats section is real and not buried. No ranking guarantees. No revenue projections. |
| Actionable without further explanation | A seller should be able to pick up the priority action plan and execute without asking follow-up questions. |

### What the report must NOT claim

- No fake client results or sales data
- No before/after ranking screenshots
- No guaranteed ranking improvements
- No fictional testimonials
- No AI-hallucinated search volume presented as fact
- No copyright-risky or trademark-risky term suggestions

## 2. Fictional Shop Concept

**Recommended shop: Maplewood Candle Co.**

A fictional Etsy candle shop selling occasion-based and personalized candles. Niches include funny retirement candles, nurse gifts, and teacher gifts. Mid-range price point ($18–$38). This is a sample SEO audit target only; the report focuses on listing visibility, not how the product is manufactured or fulfilled.

| Field | Recommendation |
|---|---|
| Shop name | Maplewood Candle Co. |
| Shop type | Etsy gift candle shop |
| Product niche | Occasion and recipient-based gift candles |
| Target buyer | Gift shoppers searching for retirement gifts, nurse gifts, teacher appreciation, funny coworker gifts |
| Example products | "Retirement Survival Kit Candle," "You're a Great Nurse" candle, teacher end-of-year gift, "Congratulations on Your Promotion" candle |
| Main SEO problem | Titles lead with cute brand phrases ("Smells Like...") instead of buyer-intent keywords. Tags repeat title words instead of adding coverage. Occasion and recipient modifiers missing from most listings. |
| Why this niche works for a sample | Buyers search by occasion + recipient + product type — easy to show structured keyword opportunity. Gift market is familiar to most buyers. Candles are relatable and visually clean for mockups. Personalization angle is present. Not tech-heavy or jargon-heavy for the buyer reading the report. |

**Why not retirement gifts mugs or teacher printables?**  
Candles sit in a familiar gift-shopping category where buyers search by recipient, occasion, and product type. That makes the SEO problems easy to demonstrate. They also have seasonal angles (holidays, end of school year) that give the report more to work with than a purely evergreen product.

### Page 5 — Current Listing Diagnosis (Shop Overview)

**Content:**

- Pattern note: "All five listings share the same root issue — title structure prioritizes brand voice over search intent."

### Pages 6–10 — Listing-by-Listing Audit (one page each)

**Purpose:** The core deliverable. Shows exactly how SearchClarity thinks about a listing.

**Content per listing:**
- Current title (quoted)
- Structural problems (bulleted, plain English)
- Keyword demand note (with data source labeled)
- Revised title (with annotation showing which words changed and why)
- Tag analysis: which tags are redundant, which are missing, gap suggestions
- Description note: what is missing, what to add
- One-line verdict

**Example verdict:** "This listing is competing on the wrong keywords. One structural change and four tag additions would put it in front of a different, higher-intent buyer."

### Page 11 — Keyword Findings

**Purpose:** Show research depth.

**Content:**
- Top 10 opportunity keywords for the shop's niche (illustrative, labeled as example data)
- Columns: keyword / search volume estimate / competition note / personalization potential / seasonal peak
- One observation per keyword cluster
- Source label on every estimate

---

### Page 12 — How We Reached These Recommendations

**Purpose:** The trust page. Show the reasoning chain so the buyer understands recommendations are not guesses.

**Content:**
1. Reviewed all five listing titles manually against Etsy search behavior patterns
2. Identified demand signals using eRank keyword tool and Etsy Marketplace Insights
3. Observed competing listings at the top of search for target keywords
4. Scored buyer-intent level per title and identified structural pattern
5. Built keyword shortlist, cross-checked with Google Trends and Pinterest Trends for seasonal shape
6. Drafted revised titles and tag suggestions against the keyword shortlist
7. Human review of all suggestions for quality, IP safety, and specificity before writing the report

---

### Page 13 — Recommended Path + Two Alternatives

**Purpose:** Give the seller a clear direction, not a pile of equal options.

**Primary path:** Title restructure + tag coverage expansion + description intent fix for all five listings.

**Alternative A — Conservative cleanup:** Fix only the two highest-opportunity listings first. Track results for 30 days. Then apply findings to remaining listings. Prioritizes safety and learning over speed.

**Alternative B — Growth test path:** Apply title and tag fixes to all five listings, then immediately add two new sub-niche listings targeting underserved long-tail keywords (e.g., "personalized nurse retirement candle" instead of generic nurse gift). Prioritizes range over depth.

---

**Purpose:** Establish professional honesty. Buyers trust people who tell them what might not work.

- No guaranteed ranking improvements
- No guaranteed sales increases

- These recommendations address visibility, not conversion — photography, price, and product quality are outside this scope
- SEO improvements take time; do not expect changes in 48 hours
- Results should be monitored and iterated on, not treated as permanent fixes

## 7. Recommendation Writing Style

Recommendations must be specific to the listing, evidence-referenced, plain English, and actionable without asking follow-up questions.

### Bad vs Good Examples

**Bad:** "Use better keywords to rank higher on Etsy."  
**Good:** "Your current title starts with a brand phrase. Etsy buyers search by occasion and recipient, not by brand name. Move 'personalized retirement gift candle' to the front, then use the remaining space for profession and year modifiers."

---

**Bad:** "Add more tags to improve visibility."  
**Good:** "Eight of your thirteen tags repeat words already in your title. Etsy treats title and tag overlap as redundant. Replace duplicates with new keyword angles: profession (nurse, teacher, RN), occasion (retirement, appreciation, end of year), and buyer context (coworker gift, office gift)."

---

**Bad:** "Optimize your listing description for SEO."  
**Good:** "Your description opens with scent notes. Buyers who reached your listing already clicked on a product — they need to know it is personalized, what name format to use, and when it ships. Move the personalization instructions to the first paragraph and push scent notes to section two."

---

**Bad:** "Target less competitive keywords."  
**Good:** "'Funny candle' has high competition and low buyer intent. 'Personalized nurse retirement gift candle' has lower competition and buyer intent baked in — the searcher is ready to buy, not browse. Your shop is set up for that buyer but not positioned to reach them."

---

**Bad:** "Make sure your title is fully optimized."  
**Good:** "Your title uses 67 of 140 allowed characters. The unused space could carry two additional modifier keywords (gift for her, candle gift set) that appear in competitor listings ranking above yours."

---

**Bad:** "This listing should perform well if you implement our suggestions."  
**Good:** "These changes address the structural barriers to visibility. Whether they improve traffic and conversion depends on many factors outside this scope, including your photos, price, and shop health."

## 8. Required Report Elements

### A — Main Recommendation

*Sample language:* "Our primary recommendation is to restructure all five listing titles to lead with buyer-intent keywords, replace redundant tags with new keyword coverage, and add occasion-specific and recipient-specific modifiers throughout. Listings 1 and 2 should be prioritized first as they represent the highest-traffic keyword opportunities in this shop."

### B — How We Reached This Recommendation

*Sample language:* "We reviewed all five listing titles against known Etsy search behavior patterns — specifically, how buyers search by occasion, recipient, and product type rather than brand phrase. We then used eRank keyword data and Etsy Marketplace Insights to confirm that buyer-intent terms for this niche have meaningful search volume and reachable competition levels for a shop at this stage. Competitor listings ranking in the top 20 for target keywords consistently front-load the occasion and recipient before any brand or descriptor language. Our title revisions reflect that pattern."

### C — Two Alternative Paths

**Conservative cleanup path:** Fix only the two highest-opportunity listings (retirement gift candle and nurse gift candle). Track impressions, click-through rate, and rank in eRank for 30 days before touching the remaining three. Prioritizes safety and learning over speed.

**Growth test path:** Apply title and tag fixes to all five listings, then immediately add two new sub-niche listings targeting underserved long-tail keywords from the keyword table (e.g., "personalized nurse retirement candle with name"). Prioritizes range over depth. Higher effort, higher potential upside.

### D — Priority Action Plan

| Stage | Actions |
|---|---|
| Do first | Restructure Listing 1 title. Rebuild Listing 2 tags. Both changes can be made in under 30 minutes. |
| Do next | Apply same title structure pattern to Listings 3 and 4. Review Listing 5 for conversion to a specific occasion. |
| Test later | Create two new listings for sub-niches identified in keyword findings. Monitor for 60 days. |
| Avoid | Do not keyword-stuff revised titles. Do not use trademarked phrases in product names. Do not use the same 13 tags across every listing in the shop. |

### E — Caveats (Sample Language)

- This report addresses listing visibility — it does not guarantee ranking improvements or sales increases.
- Etsy search rankings are influenced by many factors including recency, shop health, conversion rate, reviews, and buyer behavior that are outside the scope of listing optimization.
- Photography, pricing, and product quality affect conversion and cannot be fixed by SEO changes.
- These recommendations should be monitored and iterated on over 30–90 days, not judged after one week.
- Avoid keyword stuffing, trademark-risky phrases, and bulk-identical listings.

## 10. Professional Standards Checklist

Every real client report must pass these checks before delivery:

- [ ] Every recommendation references the specific listing, title, or tag being discussed
- [ ] Every data point from a tool is labeled with the tool name and marked as estimated or illustrative
- [ ] No ranking guarantees appear anywhere in the report
- [ ] No sales or revenue projections appear anywhere in the report
- [ ] The "how we reached this" section is present and describes the actual reasoning process
- [ ] The priority action plan can be executed without asking SearchClarity a follow-up question
- [ ] All alternatives are named, not just one path
- [ ] Caveats section is present, honest, and not buried at the back
- [ ] No keyword suggestions that reference brand names, celebrity names, trademarked phrases, or IP-risky terms
- [ ] The sample report notice is prominent on page 2 and on the cover
- [ ] Report reads like a person wrote it, not an AI content tool
- [ ] Writing does not use jargon without explanation (if "CTR" appears, it is defined)
- [ ] No copyrighted logos or branded UI screenshots used in the report
- [ ] A second human has read the full report before it is published as the sample

### Risks to Avoid

| Risk | Why it hurts | How to avoid |
|---|---|---|
| Fictional data presented as real results | Trust-destroying if a buyer notices. Legal exposure if it looks like fake testimonials. | Sample notice on every page. Label every data point. Never use "our client" language. |
| Report looks AI-generated | Undermines the "human-reviewed" positioning. Buyers are increasingly aware of AI writing patterns. | Write recommendations in direct, specific, first-person professional voice. Avoid hedge-heavy filler phrases. |
| Too long and complex for Fiverr buyers to evaluate | Buyers skim. If the value is not clear in the first two pages, they will not read further. | Executive summary and preview pages must carry the full proof load. |
| Overpromising AI or automation capabilities | Creates a gap between promise and delivery. Attracts wrong buyers. | Do not mention AI in the report or gig. Position as human-led professional analysis. |
| Using real seller listing data without permission | Privacy violation. Could be a ToS issue with Etsy. | Fictional shop only. Never use real shops without explicit written permission. |
| Keyword suggestions with IP risk | If a buyer follows a suggestion that uses a trademarked phrase, they face listing removal. SearchClarity gets blamed. | No brand names, movie/show references, celebrity names, sports teams, or viral phrases in any recommendation. |
